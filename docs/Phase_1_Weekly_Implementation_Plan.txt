# Phase 1: Weekly Implementation Plan

As a beginner, breaking down Phase 1 into a 7-day schedule will give you a concrete grip on the project. Stick to this plan day-by-day without rushing.

## Day 1: Project Setup & "Hello World" PDF
* **Objective:** Get your tools ready and verify they work.
* **Task 1:** Create a Python Virtual Environment (venv).
* **Task 2:** Install required libraries (`pip install PyMuPDF opencv-python pillow`).
* **Task 3:** Write a simple 5-line script (`test.py`) to load a CAD PDF and print the number of pages. If it prints the correct number, Day 1 is successful!

## Day 2: PDF to Image Pipeline (Task 1.1)
* **Objective:** Turn the PDF into a format the AI can actually "see".
* **Task 1:** Write a function to open the PDF using PyMuPDF (fitz).
* **Task 2:** Apply a "Zoom Factor" (Matrix) to increase the resolution to 300 DPI.
* **Task 3:** Read the first page and save it locally as `drawing_high_res.png`.

## Day 3: Mastering Basic OpenCV (Prep)
* **Objective:** Learn the basics of image manipulation.
* **Task 1:** Load the `drawing_high_res.png` using OpenCV (`cv2.imread`).
* **Task 2:** Practice converting the image from Color to Grayscale (this helps simplify shapes).
* **Task 3:** Practice basic "slicing"—crop out a random middle section of the image and save it.

## Day 4: The "Legend" Extractor (Task 1.2)
* **Objective:** Automatically isolate the symbol guide.
* **Task 1:** Use your slicing skills to crop only the right-most 20% of the `drawing_high_res.png` (since legends are usually on the right).
* **Task 2:** Save this newly cropped section as `legend_reference.png`. This is the cheat sheet for the AI.

## Day 5: Intelligent Tiling Logic (Task 1.3 Part A)
* **Objective:** Slice the massive floor plan into small grid pieces.
* **Task 1:** Large images will crash the AI. Write a simple `for` loop to divide the large image into 1000x1000 pixel chunks.
* **Task 2:** Save these chunks into a new folder called `tiles_test/`.
* **Task 3:** Open the folder and review the grid pieces.

## Day 6: The "Overlap Hack" (Task 1.3 Part B)
* **Objective:** Ensure no HVAC symbol is cut in half.
* **Task 1:** Modify your loop from Day 5. Instead of stepping 1000 pixels apart, make your loop jump by only 800 pixels.
* **Task 2:** This creates a 200px overlap. If a symbol is sliced at the edge of Tile A, it will appear fully inside Tile B.
* **Task 3:** Save these overlapping grids into a `final_tiles/` folder.

## Day 7: Integration & Review
* **Objective:** Connect everything into one clean pipeline.
* **Task 1:** Combine the code from Days 1-6 into a single file named `vision_engine.py`.
* **Task 2:** Run it from start to finish. Input: `drawing.pdf`. Output: `legend_reference.png` and a folder full of overlapping tiles.
* **Task 3:** Celebrate! You have completed the data preprocessing phase.

# Tips for Beginner Engineers
* **Print Statements:** Use `print()` liberally to check variable values and confirm where the code is currently running.
* **Documentation:** Comment your code. Write what every block does in plain English.
* **Don't Panic:** If OpenCV gives an error, it is usually because the image variables are wrong or the file path doesn't exist.
